# การตั้งค่าชื่อสินค้าหลายภาษา (Multilingual Product Names)

## ภาพรวม

ระบบ POS ตอนนี้รองรับชื่อสินค้าหลายภาษาแล้ว:
- **ภาษาไทย** (name) - ชื่อหลัก
- **ภาษาอังกฤษ** (nameEn) - ชื่อรอง
- **ภาษาลาว** (nameLo) - ชื่อรอง

เมื่อผู้ใช้เปลี่ยนภาษาในระบบ ชื่อสินค้าจะเปลี่ยนตามอัตโนมัติ

## วิธีการทำงาน

1. ระบบจะใช้ชื่อตามภาษาที่เลือก
2. ถ้าไม่มีชื่อในภาษานั้น จะ fallback ไปใช้ชื่อภาษาไทย
3. ชื่อที่แสดงจะถูกบันทึกใน transaction items ตามภาษาที่เลือกขณะขาย

## การเพิ่มข้อมูลชื่อสินค้าหลายภาษา

### วิธีที่ 1: ใช้ Script อัตโนมัติ

1. แก้ไขไฟล์ `update-product-names.js`
2. เพิ่มข้อมูลสินค้าที่ต้องการอัปเดต:

```javascript
await updateProduct('SKU', 'English Name', 'Lao Name');
```

3. รัน script:

```bash
node update-product-names.js
```

### วิธีที่ 2: อัปเดตผ่าน SQL โดยตรง

```sql
UPDATE products 
SET nameEn = 'English Name', 
    nameLo = 'Lao Name' 
WHERE sku = 'PRODUCT_SKU';
```

### วิธีที่ 3: Import จาก Excel

แก้ไขไฟล์ Excel ให้มี columns:
- `name` - ชื่อภาษาไทย
- `nameEn` - ชื่อภาษาอังกฤษ
- `nameLo` - ชื่อภาษาลาว

จากนั้น import ใหม่

## ตัวอย่างการใช้งาน

### ในหน้า POS
1. กดปุ่มเปลี่ยนภาษา (ไทย/English/ລາວ)
2. ค้นหาสินค้า (F6)
3. ชื่อสินค้าจะแสดงตามภาษาที่เลือก

### ในหน้าจอลูกค้า (Customer Display)
- ชื่อสินค้าจะแสดงตามภาษาที่เลือกในหน้า POS
- อัปเดตแบบ real-time

## หมายเหตุ

- ฐานข้อมูลจะถูกอัปเดต schema อัตโนมัติเมื่อรันแอปครั้งแรก
- ข้อมูลเดิมจะไม่สูญหาย (ชื่อภาษาไทยยังคงอยู่)
- nameEn และ nameLo เป็น optional (ไม่บังคับ)
